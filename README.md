# meteor-myboilerplate

Basic boilerplate.


=====================================
####  Version
Meteor 1.0.3.1


=====================================
####  Installation

````sh
# clone the repository
git clone https://github.com/johntday/meteor-myboilerplate.git

# change directory
cd meteor-myboilerplate

# run it
meteor
````
